Das Dialogfeld Druckereinrichtung

Mit den folgenden Optionen können Sie den Drucker und seinen Anschluß festlegen:

Drucker
Wählen Sie hier den Drucker aus, den Sie benutzen wollen. Wählen Sie entweder den "Standarddrucker", oder verwenden Sie die Option "Spezieller Drucker" und wählen dann einen der installierten Drucker aus dem Feld aus. Zum Installieren von Druckern und Einstellen von Druckeranschlüssen verwenden Sie die Systemsteuerung von Windows.
Format
Wählen Sie Hoch- oder Querformat.
Papier/Größe
Wählen Sie die Größe des Papiers, auf dem das Dokument gedruckt werden soll.
Papier/Zufuhr
Einige Drucker besitzen mehrere Schächte für verschiedene Arten der Papierzufuhr. Geben Sie hier den Schacht an.
Optionen
Zeigt ein Dialogfeld an, in dem Sie zusätzliche Auswahlmöglichkeiten haben, die spezifisch für den von Ihnen ausgesuchten Drucker sind.
Netzwerk...
Verwenden Sie diese Schaltfläche, um einem Laufwerkbuchstaben eine Verbindung zu einem Netzwerklaufwerk zuzuweisen.
