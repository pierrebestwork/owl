Der Befehl Beenden (Menü Datei)

Verwenden Sie diesen Befehl zum Beenden Ihrer Sitzung mit <<YourApp>>. Alternativ dazu können Sie aus dem Systemmenü der Anwendung den Befehl Schließen wählen. <<YourApp>> fragt nach, ob Sie Dokumente mit nicht gesicherten Änderungen speichern wollen.

Abkürzungen

Maus: Doppelklicken Sie auf das Systemmenü der Anwendung.

Tastatur: ALT+F4
