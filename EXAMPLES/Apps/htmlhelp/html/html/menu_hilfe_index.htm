Der Befehl Index (Menü Hilfe)

Verwenden Sie diesen Befehl, um den Inhaltsbildschirm der Hilfe darstellen zu lassen. Von diesem Bildschirm aus können Sie zu Anweisungen springen, die Ihnen Schritt für Schritt die Verwendung von <<YourApp>> zeigen, oder sich verschiedene Typen von Referenzinformationen anschauen.

Sobald Sie die Hilfe geöffnet haben, ist es immer möglich, die Schaltfläche Inhalt anzuklicken, um zum Inhaltsbildschirm zurückzukehren.
