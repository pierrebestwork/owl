Symbolleiste

Die Symbolleiste wird horizontal oben im Anwendungsfenster unterhalb der Menüleiste angezeigt. Sie stellt per Maus schnellen Zugriff auf viele Tools von <<YourApp>> bereit.

Um die Symbolleiste ein- oder auszublenden, können Sie aus dem Menü Ansicht den Befehl Symbolleiste auswählen (ALT,A,S).
